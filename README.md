# Tech Salary Prediction Analysis

## Overview
This project analyzes tech industry salary data and builds regression models to identify key factors influencing compensation.

## Models Used
- Linear Regression
- Ridge Regression
- Lasso Regression
- Logistic Regression (classification tasks)

## Key Features
- Data cleaning and preprocessing
- Feature engineering with demographic and experience variables
- Model comparison using R² and RMSE
- Regularization techniques to reduce overfitting
- Analysis of gender pay gap and compensation patterns

## Tech Stack
- Python
- scikit-learn
- pandas, numpy
- matplotlib

## Key Takeaways
- Applied regression techniques to real-world data
- Understood impact of different variables on salary
- Learned how regularization improves model generalization
